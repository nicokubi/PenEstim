PenEstim Application

Simulated Data

We use simulated data for the first application. Families C were simulated with a wider Weibull distribution with shape parameter, \(\alpha = 2.5\), scale parameter, \(\beta=50\), a shift of \(\delta=20\), and an asymptote parameter \(\gamma = 0.9\). This translates into an age-specific penetrance curve which does not allow for cancer age (AgeBC) to be lower than 20 with a median age of \(Q_{50}=62\) and a first quartile of \(Q_{25}=51\). The allele frequency was set at \(0.9\). Only 60 families of this type were generated to improve run-time for the simulation study using the MCMC approach.

Estimation using PenEstim

Application of the PenEstim algorithm using the clipp package for the likelihood calculation and the default prior settings.

  • First Quartile $Q_{25}$ & Scaled Beta(6,3)

  • Median $Q_{50}$ & Scaled Beta(2,2)

  • Asymptote $$ & Scaled Beta(1,1)

  • Threshold $$ & Uniform(0,25)
